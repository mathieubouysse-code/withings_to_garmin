1Taille (pour le calcul de l'IMC)

Donnée absente du CSV Withings natif
Pourquoi ce champ ?
Withings n'exporte jamais l'IMC dans le fichier weight.csv natif — seuls poids, masse grasse, masse osseuse, masse musculaire et hydratation y figurent. L'IMC est donc recalculé ici à partir de votre taille (IMC = poids ÷ taille²) pour chaque mesure.
Utilisée pour calculer l'IMC de chaque mesure de poids.

2Déposer le fichier weight.csv

Export Withings natif
Où trouver ce fichier ?
Compte Withings → Paramètres → Confidentialité & Données → Télécharger vos données → sélectionnez l'historique Poids. Le fichier s'appelle généralement weight.csv.

1Identifiants de l'application Withings

developer.withingsapi.com
Échange OAuth automatisé via un relais sécurisé
Withings n'autorise pas l'échange du code OAuth directement depuis le navigateur (le Client Secret ne doit jamais transiter côté client). Cette app utilise un petit relais (Cloudflare Worker) qui fait cet échange pour vous, en gardant le Client Secret chiffré côté serveur — aucune étape manuelle n'est nécessaire une fois le relais déployé.

Aucun proxy CORS public ne doit être utilisé pour contourner cela : votre Client Secret et votre access_token y transiteraient en clair chez un tiers que vous ne contrôlez pas. Le relais utilisé ici est déployé par vous, sur votre propre compte Cloudflare.

Si le relais n'est pas (encore) déployé ou momentanément indisponible, l'app bascule automatiquement sur un flux de secours manuel (commande curl générée et copiée dans le presse-papiers) — vous ne serez donc jamais bloqué.
